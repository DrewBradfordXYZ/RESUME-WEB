Service Task Calculator

A robust UI tool built with JavaScript and QuickBase to define and scope Professional Digital Storage Services.

1

Subject matter experts import data into QuickBase tables to generate the content and behavior of the front end UI. Document variables imported at this stage integrate with the Document Generator tool mentioned below. No coding is required to complete this step.

2

Users of the front end UI scope services by assigning task quantities, add co-provider information, additional custom roles and expenses.

3

This is the front end user interface after subject matter experts have defined the task content of the service. Notice that the task quantity calculates the level of effort (LOE) for roles assigned to the task.

Document Generator

Word Document generation tool for legal agreements spanning the globe. Users have full UI access to edit the document templates. Many factors determine which type of document is generated, which sections are used, the verbiage, as well as the formatting of the document.

1

Document section content is stored, written and editable in QuickBase. Modifications are done in a convenient WYSIWG editor that writes HTML behind the scenes. Techinical experience is not needed to modify the dynamic document content.

2

A custom legal document is generated based on service type, deal type, GEO, sales district, billing type, risk category and more. The document is attached to the bid in QuickBase and is ready to be signed by both parties.

Bid Generator

A tool that generates service bundles based on a cost factor analysis driven by simple form selections. The result is a completed bid that is ready for approval. The steps below illustrate this process.

1

Users are prompted to fill out a QuickBase form of predefined drop down selections. The Save button launches the JavaScript program that executes step 2 and 3.

2

Cost factor analysis based on the service type calculates the scope of the service. This is repeated for each associated service.

3

A complete bid is formed. Each service has milestones attached and the bid is ready for approval workflows. Users are unburdened of the multi stage cost factor analysis that go into each service, all with a simple a button click.
